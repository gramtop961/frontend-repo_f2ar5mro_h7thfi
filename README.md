A cinematic, responsive personal brand site built with Vite + React + Tailwind. It features a Spline 3D hero, parallax portrait, scroll-triggered reveals, gradient sweeps, glassmorphism social bar, glowing CTAs, timeline, portfolio grid with hover lifts, magnetic cursor, smooth transitions, scroll progress bar, lazy-loaded media, and light/dark mode. SEO includes JSON-LD Person schema.

## Tech
- Vite + React
- Tailwind CSS
- Framer Motion
- Spline React
- Lucide Icons

## Run locally
1. Install deps and start dev servers (already configured in the sandbox environment).
2. Open the frontend preview URL.

## Content
- Update `content.json` to manage portfolio items.

## Accessibility & SEO
- Semantic sections, focus-visible rings, high-contrast palette
- `application/ld+json` Person schema injected in the head

## Deployment
- Build with `npm run build` and serve `dist/` using any static host (Vercel, Netlify, Cloudflare Pages). No server required.
